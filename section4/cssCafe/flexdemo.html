Flex boxの前提知識

「Flexible Box Layout Module」。
レスポンシブなサイトを組む時にフレキシブルなレイアウトが簡単に組めるようになる
非常に便利なプロパティ群。

メニューなどを横並びに書きたいのであればfloatやinline-blockを利用して横並びにしていたが、
floatは解除方法が少し厄介であったり、
inline-blockも綺麗に整列させるには様々なプロパティの調整を しなければいけない。
このFlexBoxは横並びに整列されるためにはとても便利。 Flex boxには「Flexコンテナ」「Flexアイテム」が存在している。

display:flex

と指定すればその要素はFlexコンテナとなり、Flexコンテナの子要素は自動的にFlexアイテムとなる。
初期値として、子要素の配置を指定するflex-directionプロパティにrowという値が指定されているから横並びになる。

Floatとの違い

カラム落ちがない

floatは要素の横幅が明確でないと0と見なされてしまう。
横幅を指定するとどんな画面幅の時にも固定になってしまうため、
画面の小さい端末ではカラム落ちしていしまうレイアウトになる。

Flexxレイアウトは横幅を指定しtなくても、親要素の幅に応じてアイテムの幅を自動で
計算して均等に配置させたり、一部だけ幅固定にして後は何も指定しなくても思った通りのレイアウトができる

横並びを折り返したい flex-wrap:wrap;

初期値はflex-wrapがnowrap「折り返しなし」の状態
横並びで要素の大きさがboxの大きさを突き抜けてしまう。
flex-wrap:wrap;を指定すれば、子要素が親要素の長さを超える場合折り返すようになる。

css

  .sample{

    display:flex;

    flex-wrap:wrap;

  }

justify-contentは縦方向の揃え方を指定する。centerだと中央寄せになる.

垂直中央寄せにしたい align-items:center;

flex-directionがrowになっている場合であれば、
軸に対して垂直方向の揃え方を指定できる。
これで垂直中央寄せが簡単にできる

    display:flex;

上下左右中央寄せにしたい align-items:center;× justify-content:center;

flex-directionで定めた向きを基準として基準方向に沿った並び方を指定するのがjustify-content
基準に対して垂直方向での並び方を指定するのがalign-items
